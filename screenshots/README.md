# Extension Screenshots

This folder contains screenshots of the Watchlist Exporter browser extension in various states. These screenshots can be used when publishing the extension to browser web stores (Chrome Web Store, Firefox Add-ons, etc.).

## Screenshot Overview

### 1. Unsupported Site (`1-unsupported-site.png`)
Shows the extension popup when opened on a website that is not a supported streaming platform. Features:
- Clear message indicating supported platforms
- **Clickable hyperlinks** for each streaming platform
- Direct links to watchlist pages (not homepages)
- Helpful call-to-action encouraging users to try the extension

### 2. Navigate to Watchlist (`2-navigate-to-watchlist.png`)
Shows the extension popup when on a supported streaming site but not on the watchlist page. Features:
- Warning that user needs to navigate to watchlist page first
- Platform-specific watchlist URL as a clickable link
- Note about lazy loading behavior

### 3. Ready to Export (`3-ready-to-export.png`)
Shows the extension popup when on a watchlist page, ready to export. Features:
- Clear instructions
- Prominent "Export Watchlist" button
- Clean, simple interface

### 4. Exporting (`4-exporting.png`)
Shows the extension popup during the export process. Features:
- Disabled button showing "Exporting..." state
- Progress message "Extracting watchlist data..."
- Visual feedback for the user

### 5. Export Success (`5-export-success.png`)
Shows the extension popup after a successful export. Features:
- Success message with count of exported items
- Green success indicator
- Re-enabled export button for additional exports

## Supported Platforms

As of the current version, the extension supports:
- Netflix
- Prime Video
- Disney+
- Hulu
- Apple TV+
- Max
- Peacock
- Paramount+

## Usage for Publishing

When publishing to browser extension stores, you can use these screenshots to:
1. Show the extension's user interface
2. Demonstrate the workflow
3. Highlight the new clickable platform links feature
4. Display all supported streaming platforms

## Dimensions

All screenshots are taken at the extension's actual popup size (320px width) with appropriate padding for display purposes.
